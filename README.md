## willparken-web - Starten

1. Clone das Repository
   ```sh
   git clone https://github.com/saidov-arb/willparken-web.git
   ```

2. NodeModules, etc. herrichten
   ```sh
   npm install
   ```

3. API Starten
    ```sh
   npm run serve
   ```
